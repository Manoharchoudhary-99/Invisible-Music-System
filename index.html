About Project

This project introduces a touchless control experience using an ultrasonic sensor connected to ESP8266. Hand gestures are translated into commands for audio mode switching and volume control, enabling safe and intuitive interaction in smart spaces, assistive systems, and modern IoT environments.

The innovation lies in combining low-cost embedded hardware with a futuristic interface: users interact naturally, while the system instantly responds through OLED updates and buzzer tones.
